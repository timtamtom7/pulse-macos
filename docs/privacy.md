# Pulse Privacy Policy

**Last updated: 2026-03-25**

## Overview

Pulse is a menu bar system monitor for macOS. All statistics are read locally and no personal data is collected.

## Data Collection

Pulse reads macOS system statistics only:
- CPU usage
- RAM usage
- Disk usage
- Network activity
- Temperature (CPU, GPU, battery)
- Process information

**No personal data is collected, stored externally, or transmitted to any server.**

## Local Storage

All sampled data stays in `~/Library/Application Support/Pulse/pulse.db` on your Mac.

## iCloud Sync

If enabled, only user preferences sync via iCloud Key-Value store:
- Settings and preferences only
- No statistics or personal data
- User can disable anytime in Settings

## Widgets

Widget data stays on-device:
- Widget extension reads from app group container only
- No third-party data sharing
- No analytics from widgets

## Analytics

Pulse uses no third-party analytics SDKs.

Optional anonymous crash reporting via Apple-bundled CrashReporter (App Store distribution).

## Third Parties

Pulse does not include:
- Analytics SDKs
- Ad networks
- Data brokers
- Any third-party services

## Children

Pulse is not directed at users under 13.
